## GATE: Conceitos Fundamentais

### Step 1: Conceitos-Norte Internalizados

- [ ] Entendo que transformacao e o norte (nao formato, nao bonus)
- [ ] Entendo que provas servem tambem DENTRO do produto (nao so para vender)
- [ ] Entendo que melhoria e continua (lancar e melhorar)
- [ ] Entendo overdeliver (entregar mais do que prometeu)
- [ ] Entendo a importancia da jornada de sucesso visivel

---

### Step 2: Jornada de Sucesso do Cliente

- [ ] Ponto de partida definido (onde o cliente esta hoje)
- [ ] Ponto de chegada definido (transformacao prometida)
- [ ] Etapas intermediarias mapeadas (marcos de progresso)
- [ ] Sistema de visualizacao de progresso criado (faixas, niveis, badges)
- [ ] Cada marco representa progresso real (nao artificial)

---

## ESCOLHA DE FORMATO

### Step 3: Formato Escolhido

- [ ] Formato definido: ( ) Curso ( ) Mentoria ( ) Mastermind ( ) Consultoria ( ) Peixe Assado
- [ ] Justificativa documentada (escala vs margem vs resultado)
- [ ] Precificacao relativa entendida (curso = base)

---

## ESTRUTURACAO DO PRODUTO

### Step 4: Estrutura do Curso Online

**Passo 1 — Ponto de partida e chegada:**
- [ ] Jornada de sucesso serve como base (Step 2)

**Passo 2 — Macro etapas:**
- [ ] Jornada dividida em 2-7 macro etapas (modulos)
- [ ] NAO mais que 7 modulos

**Passo 3 — Subetapas:**
- [ ] Cada macro etapa subdividida em 2-7 subetapas

**Passo 4 — Unidades de aprendizado:**
- [ ] Subdivisao continuou ate chegar em unidades faceis de aprender (aulas)

**Passo 5 — Objecoes:**
- [ ] Objecoes de aplicacao mapeadas
- [ ] Para cada objecao: prova ou argumento logico para matar
- [ ] Provas de alunos incluidas dentro do curso (nao so na venda)

**Guardrails:**
- [ ] Ensinando o MINIMO necessario para gerar transformacao
- [ ] NAO encheu de conteudo desnecessario
- [ ] Perguntas de alunos sao publicas (inteligencia coletiva)

---

### Step 5: Aula Inaugural

- [ ] Boas-vindas incluidas
- [ ] Reconhecimento da decisao certa incluido (acalmar inseguranca)
- [ ] Tour pelo curso incluido (como funciona portal, comunidade, suporte)
- [ ] Jornada de sucesso mostrada (marcos, faixas, tempo esperado)
- [ ] Formato escolhido: ( ) Ao vivo ( ) Gravada

---

### Step 6: Modulos Extras (Bonus)

- [ ] Bonus tipo 1: Mata objecao → _________________
- [ ] Bonus tipo 2: Resolve problema futuro → _________________
- [ ] Bonus tipo 3: Acelera transformacao → _________________

**Guardrails:**
- [ ] Nenhum bonus e algo que deveria estar no curso principal
- [ ] Bonus de venda separados de bonus surpresa (overdeliver)

---

### Step 7: Onboarding

- [ ] Acesso enviado por e-mail
- [ ] Acesso enviado por SMS
- [ ] Acesso enviado por WhatsApp
- [ ] Verificacao de quem comprou e NAO acessou o portal
- [ ] Ligacao para nao-acessores (se possivel, no primeiro dia)

---

### Step 8: Estrutura da Mentoria (se aplicavel)

- [ ] Roma definida (igual ou diferente do curso, com continuidade)
- [ ] Entregaveis definidos:
  - [ ] Navegacao ativa (acompanhamento proativo)
  - [ ] Analise de implementacao
  - [ ] Acoes especificas / GPS
  - [ ] Sprints de resultado / desafios
  - [ ] Eventos de networking
- [ ] Calendario anual de entregaveis criado
- [ ] Posicionamento: "ate chegar ao objetivo" (nao escola)
- [ ] SEM garantia (mentoria exige comprometimento)

---

### Step 9: Estrutura do Mastermind (se aplicavel)

- [ ] Pre-requisito de entrada definido (ex: faixa preta)
- [ ] Processo de curadoria/entrevista criado
- [ ] 3 encontros anuais planejados (2 dias cada)
- [ ] Comunidade de negocios criada (Facebook ou similar)
- [ ] Comunidade social criada (Telegram)
- [ ] Regra comunicada: participacao e OBRIGATORIA (contribuir)

---

## PRECIFICACAO

### Step 10: Preco Definido

**Oportunidade:**
- [ ] Atratividade da Roma avaliada
- [ ] Escassez definida (quando abre, por quanto tempo)

**Risco:**
- [ ] Provas catalogadas: ( ) Muitas ( ) Poucas ( ) Nenhuma
- [ ] Se poucas/nenhuma provas: garantia forte definida
- [ ] Se muitas provas: garantia simples suficiente
- [ ] Provas mostram SEMELHANCA com publico-alvo (diversidade)

**Preco:**
- [ ] Formato: ( ) Curso (base) ( ) Mentoria (5-10x) ( ) Mastermind (10-20x) ( ) Consultoria (10x+) ( ) Peixe Assado (livre)
- [ ] Preco definido: R$ _______________
- [ ] Garantia definida: _______________
