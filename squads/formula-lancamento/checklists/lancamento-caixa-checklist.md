## GATE: Pré-Requisitos

### Step 1: Validação de Pré-Requisitos

**Pré-requisito 1 — Lista:**
- [ ] Tenho uma lista de e-mails (prospectos, leads ou clientes)
- [ ] A lista tem tamanho suficiente para justificar o esforço

**Pré-requisito 2 — Produto Estabelecido:**
- [ ] Tenho um produto/serviço que JÁ é vendido no mercado
- [ ] Minha audiência já SABE que o produto existe
- [ ] Minha audiência já SABE o preço do produto

**Decision:**
- [ ] PASS: Ambos pré-requisitos atendidos → prosseguir
- [ ] VETO: Sem lista ou sem produto estabelecido → NÃO fazer este lançamento

---

## PLANEJAMENTO

### Step 2: Período Definido

**Execução:**
- [ ] Escolhi o período de 4 dias consecutivos
- [ ] Opção: ( ) Seg-Qui ( ) Ter-Sex ( ) Outro (justificado)

**Guardrail:**
- [ ] Lançamento NÃO abre nem fecha no final de semana (exceto se conheço muito bem meu mercado)
- [ ] Datas marcadas no calendário

---

### Step 3: Decisão sobre Antecipação

- [ ] Decidi: ( ) COM antecipação (e-mail na semana anterior) ( ) SEM antecipação
- [ ] Se COM: e-mail de antecipação escrito com história + razão

---

## PREPARAÇÃO

### Step 4: Oferta Exclusiva Montada

**Forma de exclusividade escolhida:**
- [ ] ( ) Desconto real sobre preço conhecido
- [ ] ( ) Bônus exclusivo (algo que não pode ser comprado normalmente)
- [ ] ( ) Consultoria/Serviço que normalmente não ofereço
- [ ] ( ) Frete grátis (produto físico)
- [ ] ( ) Combinação de formas acima

**Guardrails:**
- [ ] A oferta PARECE exclusiva para a audiência (não só para mim)
- [ ] Se desconto: audiência já conhece o preço real
- [ ] Se bônus: NÃO é algo que ofereço o tempo inteiro
- [ ] Oferta não foi usada recentemente (efeito antibiótico)

**Estratégia high ticket (opcional):**
- [ ] ( ) Bônus de consultoria/serviço como porta para venda high ticket
- [ ] ( ) Não aplicável

---

### Step 5: Razão Definida (Gatilho Mental Rei)

**Razão escolhida:**
- [ ] ( ) Black Friday
- [ ] ( ) Aniversário (pessoal ou da empresa)
- [ ] ( ) Data comemorativa relevante ao mercado
- [ ] ( ) Queima de estoque
- [ ] ( ) Conta inesperada
- [ ] ( ) Agradecimento
- [ ] ( ) Luta por causa
- [ ] ( ) Virada de lote (evento)
- [ ] ( ) Outra razão genuína: _______________

**Validação:**
- [ ] A razão é genuína (não inventada)
- [ ] A razão responde claramente: "por que estou fazendo essa oferta AGORA?"
- [ ] A razão é crível para a audiência

---

### Step 6: História Construída

- [ ] Transformei a razão em uma narrativa
- [ ] A história conta como cheguei na decisão de fazer a oferta
- [ ] A história conecta emocionalmente com a audiência

---

### Step 7: Bônus Reservado para E-mail 3

- [ ] Separei um bônus que NÃO será anunciado nos e-mails 1 e 2
- [ ] O bônus é atrativo o suficiente para ser novidade no dia 3
- [ ] Quem comprou nos dias 1-2 também vai ganhar o bônus (nota no e-mail)

---

### Step 8: 4 E-mails Escritos

#### E-mail 1 (Dia 1): Abertura
- [ ] Abre com história (de onde veio a razão)
- [ ] Descreve a oferta exclusiva completa
- [ ] Informa o prazo (até dia 4)
- [ ] CTA com link (checkout direto ou página de vendas)
- [ ] Gatilho da razão presente
- [ ] Gatilho da escassez presente (prazo)

#### E-mail 2 (Dia 2): Conteúdo
- [ ] Abre com conteúdo de valor (ensino)
- [ ] Lembrete breve: oferta + prazo + CTA
- [ ] Seção "pra quem não leu o e-mail de ontem" → repete copy do E-mail 1

#### E-mail 3 (Dia 3): Bônus Surpresa
- [ ] Anuncia o bônus reservado ("o que estava bom ficou melhor")
- [ ] Nota: quem já comprou também ganha
- [ ] Breve descrição da oferta + CTA
- [ ] Seção para quem não leu antes → repete copy do E-mail 1

#### E-mail 4 (Dia 4): Última Chamada
- [ ] Abre com tom de última chamada — termina HOJE
- [ ] CTA direto — não deixa pra depois
- [ ] Seção para quem não leu antes → repete copy do E-mail 1

**Validação global dos e-mails:**
- [ ] Todos os e-mails têm o gatilho da razão
- [ ] Todos os e-mails têm escassez (prazo)
- [ ] E-mails 2, 3 e 4 incluem seção de recap para quem não leu os anteriores

---

### Step 9: Preparação Técnica

**Checkout/Página:**
- [ ] Checkout configurado (ou página de vendas com vídeo, se oferta complexa)
- [ ] Link testado e funcionando

**E-mail Marketing:**
- [ ] E-mails formatados dentro do sistema de e-mail marketing
- [ ] E-mail teste enviado para mim mesmo
- [ ] Tags e segmentação corretas
- [ ] E-mails agendados OU prontos para disparo manual

**Guardrail:**
- [ ] Tudo preparado ANTES do Dia 1 (não no dia do disparo)
- [ ] Horário de disparo definido: primeira hora da manhã (6h-8h)

---

## EXECUÇÃO (4 Dias)

### Step 10: Disparar e Monitorar

**Dia 1:**
- [ ] E-mail 1 disparado na hora certa
- [ ] Links verificados e funcionando
- [ ] Vendas sendo acompanhadas

**Dia 2:**
- [ ] E-mail 2 disparado
- [ ] Links verificados

**Dia 3:**
- [ ] E-mail 3 disparado (bônus surpresa)
- [ ] Links verificados

**Dia 4:**
- [ ] E-mail 4 disparado (última chamada)
- [ ] Links verificados
- [ ] Monitoramento intensivo (70-80% das vendas vem hoje)

**Guardrail Emocional:**
- [ ] NÃO desanimei por vendas baixas nos dias 1-3 (a curva é assim mesmo)
- [ ] NÃO deixei de mandar e-mail por achar que não estava funcionando
- [ ] Executei todos os 4 dias até o final

---

### Step 11: Fechamento Real

- [ ] Oferta encerrada no horário anunciado
- [ ] Checkout/página de vendas desativado
- [ ] NÃO prolonguei nem reabri a oferta

**Guardrail:**
- [ ] Se alguém pediu para comprar depois → NÃO reabri
- [ ] Fechamento foi REAL (mercado aprende se não for)

---
