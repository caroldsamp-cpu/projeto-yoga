## BLOCO A: R2X — Geracao de Testemunhos

### Step 1: Pre-requisitos do R2X

- [ ] Produto foi entregue e alunos ja aplicaram
- [ ] Ha alunos com transformacao real (chegaram na Roma)
- [ ] Premios definidos (bons E dentro da realidade financeira)
- [ ] Plataforma de recebimento de videos definida

---

### Step 2: Etapa Concurso — 3 Emails em 7 Dias

**Email 1 — Anuncio (Dia 1):**
- [ ] Abre com PROMESSA (premio no titulo)
- [ ] Anuncia o concurso
- [ ] Explica a RAZAO do concurso
- [ ] Detalha TODOS os premios (participacao + top 3 + melhor)
- [ ] Explica COMO participar
- [ ] Regra obrigatoria: video fala da ROMA/transformacao
- [ ] Regra obrigatoria: maximo 3 minutos
- [ ] Regra obrigatoria: gravar na horizontal
- [ ] Criterios de aceitacao claros (fora = nao elegivel)
- [ ] Clausula de autorizacao de imagem incluida
- [ ] Metodo de selecao explicado (equipe ou votacao)
- [ ] Data do anuncio dos ganhadores informada
- [ ] Prazo definido (~7 dias)

**Email 2 — Lembrete (2 dias antes):**
- [ ] Lembrete com TODAS as informacoes do Email 1 repetidas

**Email 3 — Ultimo dia:**
- [ ] "Termina hoje as 23:59" + todas as informacoes repetidas

**Guardrails:**
- [ ] NAO enviou email faltando menos de 2h pro prazo
- [ ] NAO deu prazo maior que 14 dias

---

### Step 3: Etapa Selecao + Divulgacao

- [ ] Metodo de selecao definido: ( ) Equipe interna ( ) Votacao publica
- [ ] Se equipe: criterios de avaliacao definidos previamente
- [ ] Se votacao: top 10 publicados + prazo + regras de votacao
- [ ] Ganhadores divulgados PUBLICAMENTE
- [ ] Divulgacao no DIA prometido (transparencia)

## BLOCO B: Encontrar e Fechar com Expert

### Step 5: Metodo LEOU

**L — Listar:**
- [ ] 28 potenciais experts listados
- [ ] Fontes diversas usadas (Google, Instagram, YouTube, livros, podcasts)

**E — Examinar:**
- [ ] Cada expert avaliado por RESULTADO (tem transformacao real?)
- [ ] Cada expert avaliado por AUDIENCIA (seguidores + engajamento)
- [ ] Lista refinada para os melhores candidatos

**O — Oferta Irresistivel:**
- [ ] Promessa definida (faixa preta / resultado significativo)
- [ ] Provas preparadas (casos de sucesso documentados)
- [ ] Historia de procedencia escrita
- [ ] Oferta financeira definida (0 investimento + 100% 1o lanc + 50/50 depois)

**U — Upar (Abordar):**
- [ ] Forma de abordagem definida (DM, email, indicacao)
- [ ] Frequencia de follow-up definida
- [ ] Abordagens iniciadas

### Step 13: Acompanhamento e Otimizacao

**Acompanhamento DIARIO (leads):**
- [ ] Perguntou: quantas leads ontem?
- [ ] Perguntou: qual CPL?
- [ ] Perguntou: quanto investido / quanto falta?
- [ ] Perguntou: % da meta atingida?

**Acompanhamento QUINZENAL (distribuicao):**
- [ ] Relatorio recebido com visualizacoes por tipo e plataforma
- [ ] Custo por visualizacao medio acompanhado
- [ ] Tendencia identificada (subindo = problema / baixando = bom)
- [ ] Atividades de otimizacao do gestor verificadas

**Se CPL caro — verificou as 3 alavancas:**
- [ ] Criativos: gestor testou novos anuncios?
- [ ] Publicos: gestor testou novos publicos/segmentacoes?
- [ ] Orcamento: gestor redistribuiu $ dos publicos caros para baratos?

**Vitimometro:**
- [ ] Gestor traz solucoes (NAO so desculpas)
- [ ] Gestor NAO diz "nao tem mais o que fazer" (possibilidades sao infinitas)
