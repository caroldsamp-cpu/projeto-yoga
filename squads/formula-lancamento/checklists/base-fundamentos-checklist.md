## PARTE 1: Fundamentos Estratégicos

### Step 1: Nicho Definido

**Precondições:**
- [ ] Nenhuma precondição — este é o ponto de partida

**Execução:**
- [ ] Identifiquei o mercado/área onde vou atuar
- [ ] Avaliei se o nicho é grande demais (tem concorrentes gigantes?)
- [ ] Apliquei subnicho para reduzir a concorrência direta
- [ ] Nicho tem espaço para uma Roma única e diferenciada?

**Output Validado:**
- [ ] `nicho_definido.txt` criado
- [ ] Nicho e subnicho claramente documentados no arquivo

**Guardrail:**
- [ ] Nicho NÃO é tão amplo que compete com faculdades/agências gigantescas
- [ ] Nicho NÃO é tão estreito que inviabiliza audiência suficiente para 6em7

---

### Step 2: Avatar Definido

**Precondições:**
- [ ] Nicho definido (Step 1 completo)

**Execução:**
- [ ] Descrevi o avatar em detalhe: quem é, o que faz, o que sente
- [ ] Identifiquei dores, medos, desejos e obstáculos do avatar
- [ ] Mapeei o vocabulário que o avatar usa para descrever seus problemas
- [ ] Perfil documentado em formato escrito

**Output Validado:**
- [ ] `avatar_profile.txt` criado
- [ ] Perfil é uma pessoa específica, não "público geral"

---

### Step 3: Roma Definida

**Precondições:**
- [ ] Avatar definido (Step 2 completo)

**Execução:**
- [ ] Formulei a transformação prometida em linguagem do avatar (não técnica)
- [ ] Roma está em 1-2 frases concisas
- [ ] Testei: a Roma é replicável com um método?
- [ ] Testei: qualquer aluno determinado consegue chegar nessa Roma?

**Teste de Qualidade da Roma:**
- [ ] Consigo responder "Como esse conteúdo ajuda o avatar a chegar em Roma?" para qualquer conteúdo que criar?
- [ ] Consigo responder "Essa copy leva o avatar mais perto de Roma?" para qualquer copy que criar?
- [ ] Roma NÃO depende de talento único irreplicável (ex: "se tornar o Cristiano Ronaldo")

**Output Validado:**
- [ ] `roma_definida.txt` criado
- [ ] Roma é atrativa para o avatar definido no Step 2

---

### Step 4: Método Formalizado

**Precondições:**
- [ ] Roma definida (Step 3 completo)

**Execução:**
- [ ] Mapeei o passo a passo do avatar atual → Roma
- [ ] Dei nome ao método
- [ ] Documentei em formato escrito (não está só na minha cabeça)
- [ ] Qualquer pessoa determinada consegue replicar?

**Output Validado:**
- [ ] `metodo_formalizado.txt` criado
- [ ] Método NÃO depende de sorte ou talento especial para funcionar

## PARTE 2: Oferta

### Step 5: Oferta Montada

**Precondições:**
- [ ] Método formalizado (Step 4 completo)

#### 5.1 Entregáveis

**Execução:**
- [ ] Formato do veículo definido (curso online / consultoria / mentoria / físico / serviço)
- [ ] Conteúdo dos módulos esboçado (o que vai ter dentro do produto)
- [ ] Quantidade de módulos / aulas esboçada

**Output Validado:**
- [ ] Formato mais eficiente para 6em7 escolhido (preferência: curso online)

#### 5.2 Preço

**Execução:**
- [ ] Identificado comparável: quanto custa o avatar NÃO ter a Roma?
- [ ] Preço definido em valor que faz sentido financeiro para mim
- [ ] Formas de pagamento definidas (parcelas, boleto, cartão)

**Output Validado:**
- [ ] Preço > zero + formas de pagamento documentadas

**Guardrail:**
- [ ] IF Roma é única → NÃO preciso me comparar com concorrentes no preço
- [ ] IF Roma é genérica → levei em conta preços do mercado

#### 5.3 Garantia

**Execução:**
- [ ] Política de garantia definida

**Output Validado:**
- [ ] Garantia documentada

#### 5.4 Bônus (opcional)

**Execução:**
- [ ] Lista de bônus definida (ou decidiu não ter bônus)

**Output Validado:**
- [ ] `oferta_completa.txt` criado com Entregáveis + Preço + Garantia + Bônus

## PARTE 3: Audiência

### Step 6: Sistema de Conteúdo Raiz Criado

**Precondições:**
- [ ] Roma definida (Step 3 completo)

#### 6.1 Roteiro de Conteúdo

**Execução:**
- [ ] Entendi que o papel do lançador no conteúdo é conectar o "como" à Roma ANTES de ensinar
- [ ] Cada conteúdo começa conectando a técnica ensinada com a Roma do avatar
- [ ] NÃO estou ensinando só o "como" sem mostrar por que isso leva o avatar à Roma

**Checagem rápida antes de publicar qualquer conteúdo:**
- [ ] Esse conteúdo começa mostrando como ajuda o avatar a chegar em Roma?
- [ ] Só então o "como" técnico aparece depois?

#### 6.2 Linhas Editoriais

**Execução:**
- [ ] Defini 2-3 linhas editoriais para o meu nicho
- [ ] Cada linha tem formato pré-definido + tema central
- [ ] Não vou criar conteúdo "de folha em branco" — tenho formato definido

**Output Validado:**
- [ ] `linhas_editoriais.txt` criado com 2-3 linhas documentadas

#### 6.3 Consistência

**Execução:**
- [ ] Defini frequência semanal de publicação (mínimo: 2x/semana)
- [ ] Frequência é realista para eu manter consistentemente

**Guardrail:**
- [ ] Consistência mínima de 2x/semana > publicar 8x numa semana e sumir 3 semanas

#### 6.4 Ferramentas de Ensino

**Execução:**
- [ ] Incluindo histórias sempre que possível no conteúdo
- [ ] Usando analogias para explicar conceitos complexos
