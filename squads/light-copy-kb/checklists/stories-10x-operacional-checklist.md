## Parte 2: SPIN Selling no Inbox

### Step 2.1: Triagem de Leads [GATE]

- [ ] Revisando inbox diariamente
- [ ] Priorizando leads por qualificacao:
  - [ ] Texto longo e detalhado = ALTA prioridade
  - [ ] Objecoes especificas = MUITO ALTA (ja considerando compra)
  - [ ] Perguntas sobre produto/preco = PRONTO PARA COMPRAR
  - [ ] Emojis genericos / "legal" = BAIXA — nao priorizar
- [ ] Leads quentes identificados: _____ (quantidade)

**GATE:** Sem triagem = tempo gasto com leads frios enquanto quentes esfriam.

> Fonte: [SOURCE: S10X-Operacional > Inbox lucrativo]

### Step 2.2: Conversa SPIN

Para cada lead quente, verificar:

- [ ] **S — Situacao:** Fiz perguntas de diagnostico
  - "Qual e seu [metrica] hoje?"
  - "Ha quanto tempo voce [situacao]?"
- [ ] **P — Problema:** Cavei a dor
  - "Que [solucao] ja tentou? Por que nao funcionou?"
- [ ] **I — Implicacao:** Amplifiquei urgencia
  - "O que acontece se isso continuar por 5 anos?"
- [ ] **N — Necessidade:** Levei a conclusao
  - "Voce acredita que deveria investir mais nisso?"
- [ ] **Hook Question:** Pedi a acao
  - "Sabendo de tudo isso, qual a conclusao mais obvia?"
- [ ] Se comprou → pedi comprovante
- [ ] Se nao comprou → ofereci: bonus / call / trial 7 dias

### Step 2.3: Anti-Patterns Verificados

- [ ] NAO empurrei produto sem diagnostico SPIN
- [ ] NAO tive medo de vender (nao congelei no fechamento)
- [ ] NAO insisti demais apos "nao" claro
- [ ] Apliquei Pressuposicao do Sim (agi como se venda estivesse acontecendo)

> Fonte: [SOURCE: S10X-Operacional > Inbox lucrativo]

## Parte 3: Crescimento de Seguidores

### Step 3.1: Conteudo para Impulsionamento

- [ ] Publicacao criada (carrossel, foto, ou video)
- [ ] Conteudo e do NICHO (nao generico)
- [ ] Entendo que taxa de acerto e ~10 de 150 posts — testando volume

### Step 3.2: Configuracao do Impulsionamento [GATE]

- [ ] Usando botao nativo "Impulsionar" do Instagram (NAO Meta Ads Manager)
- [ ] Objetivo: **Visitas ao Perfil**
- [ ] NAO selecionei Engajamento
- [ ] NAO selecionei Cliques no Link
- [ ] Audiencia configurada:
  - [ ] Genero definido
  - [ ] Idade: 18-65
  - [ ] Localizacao definida
  - [ ] Interesses relevantes ao nicho
- [ ] Duracao: 7 dias
- [ ] Orcamento diario: R$_____ (min R$10; recomendado R$100)

**GATE:** Objetivo errado = dinheiro desperdicado. DEVE ser Visitas ao Perfil.

> Fonte: [SOURCE: S10X-Operacional > Crescimento de seguidores]

### Step 3.3: Rastreamento

- [ ] Registrado no App S10X:
  - [ ] Nome da publicacao
  - [ ] Formato
  - [ ] Data
  - [ ] Audiencia
  - [ ] Orcamento
  - [ ] Seguidores no inicio: _____
  - [ ] Seguidores no fim: _____
  - [ ] Custo por seguidor: R$_____
- [ ] Custo por seguidor < R$1? (ideal R$0.30-0.80)
  - [ ] Se > R$2 → conteudo nao e interessante → criar novo
- [ ] Se audiencia ficou cara → testar outra segmentacao

> Fonte: [SOURCE: S10X-Operacional > Crescimento de seguidores]

## Parte 4: Trafego Perpetuo (se aplicavel)

### Step 4.1: Estrutura de Campanhas

- [ ] Pagina de vendas pronta e funcionando
- [ ] Pixel instalado e rastreando
- [ ] 1 campanha CBO configurada com:
  - [ ] Ad Set HOT (compradores, checkout, engajados)
  - [ ] Ad Set WARM (visitantes, viewers, email)
  - [ ] Ad Set COLD (lookalikes, interesses)
- [ ] 3 criativos por ad set
- [ ] Minimo de gasto definido por ad set

> Fonte: [SOURCE: S10X-Operacional > Perpetuo]

### Step 4.2: Distribuicao de Budget

- [ ] Nivel definido (marque UM):
  - [ ] **Iniciante:** 95% conversao + 5% remarketing
  - [ ] **Avancado:** 15% descoberta + 10% relacionamento + 60% conversao + 5% remarketing + 10% reserva
- [ ] Budget total diario: R$_____

### Step 4.3: Remarketing por Tempo

- [ ] Remarketing segmentado por janela temporal:
  - [ ] Dias 1-7: Bonus, garantia, depoimentos
  - [ ] Dias 8-10: Flash sales, aula densa
  - [ ] Dias 11-13: Descontos, prova social
  - [ ] Dias 14-15: Push final
- [ ] Cada periodo EXCLUI audiencia do periodo anterior

### Step 4.4: Metricas Perpetuo

- [ ] Rastreando metricas secundarias: Alcance, Frequencia (<4x), CTR (>2% = bom)
- [ ] Rastreando metrica primaria: CPA (meta: metade do valor do produto)
- [ ] Rastreando nos de conversao: Cliques → Visitas → Checkout → Vendas
- [ ] NAO pausei perpetuo durante lancamento

> Fonte: [SOURCE: S10X-Operacional > Perpetuo]
