## Parte 1: Narrativa de Premissas

### Step 1.1: Definir Conclusao Desejada [GATE]

- [ ] Conclusao de venda identificada (o que o leitor deve concluir SOZINHO)
- [ ] Conclusao NAO e dita diretamente no texto
- [ ] Conclusao NAO e "compre meu produto" — e uma transformacao que o leitor deduz
- [ ] Briefing de copy do M2 (Step 1.1) disponivel como input

**GATE:** Sem conclusao definida = sem premissas. A conclusao e o destino — sem ela, nao ha caminho.

> Fonte: [SOURCE: M03 > Fundamentos da Logica das Premissas]

### Step 1.2: Construir Cadeia de Premissas [GATE]

- [ ] 3-5 premissas escritas
- [ ] Cada premissa e DESPRETENSIOSA (nao revela intencao de venda)
- [ ] Cada premissa verificada contra 6 criterios:

| Criterio | P1 | P2 | P3 | P4 | P5 |
|----------|----|----|----|----|-----|
| Curiosa? | [ ] | [ ] | [ ] | [ ] | [ ] |
| Factual? | [ ] | [ ] | [ ] | [ ] | [ ] |
| Aferivel (verificavel)? | [ ] | [ ] | [ ] | [ ] | [ ] |
| Especifica (numeros/nomes)? | [ ] | [ ] | [ ] | [ ] | [ ] |
| Irrefutavel? | [ ] | [ ] | [ ] | [ ] | [ ] |
| Polemica? (opcional) | [ ] | [ ] | [ ] | [ ] | [ ] |

- [ ] Li premissas para alguem — ninguem refutou
- [ ] Nenhuma premissa revela o produto diretamente

**GATE:** Premissas pretensiosas = copy de promessa, NAO Light Copy. Reescreva ate serem despretensiosas.

> Fonte: [SOURCE: M03 > Encontrando Premissas]

### Step 1.3: Sequenciar Premissas

- [ ] Premissas ordenadas em sequencia logica progressiva
- [ ] Cada premissa constroi sobre a anterior
- [ ] Teste: leitor chega na conclusao ANTES de voce dizer?
- [ ] NAO pulei para a conclusao cedo demais

> Fonte: [SOURCE: M03 > Analisando narrativas reais]

## Parte 2: Selecao de Elementos Literarios

### Step 2.1: Identificar Contexto e Objetivo

- [ ] Objetivo principal da peca definido (marque UM):
  - [ ] Anuncio de conversao
  - [ ] Viralidade/views
  - [ ] Vendas/pitch
  - [ ] Ideias originais
  - [ ] Comunidade/tribo
  - [ ] Humor
- [ ] Consultei tabela de elementos prioritarios por contexto
- [ ] Elementos prioritarios para meu contexto identificados

> Fonte: [SOURCE: M03 > Gancho visual] + [SOURCE: M03 > Antitese] + [SOURCE: M03 > Confianca extrema]

### Step 2.2: Selecionar 3-5 Elementos Complementares

- [ ] 3-5 elementos selecionados
- [ ] Elementos se combinam/potencializam mutuamente
- [ ] Justificativa para cada elemento documentada
- [ ] OPCIONAL: Consultei combinacoes classicas:

| Combinacao | Usado? |
|-----------|--------|

## Parte 3: Aplicacao dos Elementos

### Step 3.1: Aplicar Elementos ao Texto [GATE]

- [ ] Texto base disponivel (copy do M2 ou rascunho)
- [ ] Esqueci a venda primeiro — criei a situacao
- [ ] Cada elemento selecionado foi aplicado ao texto

**Elementos aplicados (marque os que usou):**

| Elemento | Aplicado? | Anti-Pattern Verificado? |
|----------|-----------|------------------------|
| Aforismo | [ ] | [ ] Nao e frase generica — e universal e memoravel |
| Antitese | [ ] | [ ] Nao forcei contraste — e natural e justo |
| Anafora | [ ] | [ ] Ultimo item QUEBRA o padrao |
| Apelo Cotidiano | [ ] | [ ] Objeto e do dia-a-dia real, nao rebuscado |
| Autodepreciacao | [ ] | [ ] Tenho momento de autoridade ANTES |
| Confianca Extrema | [ ] | [ ] Nao confundi com arrogancia — sem simpatia exagerada |
| Desfecho Inesperado | [ ] | [ ] NAO anunciei que haveria surpresa |
| Easter Egg | [ ] | [ ] NAO expliquei a referencia — so insider entende |
| Gancho Visual | [ ] | [ ] Primeiros 2-4 seg com IMAGEM/OBJETO fisico |
| Hiperbole | [ ] | [ ] Exagero e ESCARRADO (obvio que e recurso) |
| Jogo de Palavras | [ ] | [ ] Duplo sentido funciona sem forcar |
| Linguagem Sensorial | [ ] | [ ] Usei 2+ sentidos (texturas, sons, cheiros) |
| Linguagem Fantasiosa | [ ] | [ ] Criei cenario impossivel a partir de "como seria se..." |
| Lista | [ ] | [ ] 5-10 itens relacionados, nao menos |
| Mundo ao Contrario | [ ] | [ ] Inversao e clara e tem proposito |
| Neologismo | [ ] | [ ] Palavra inventada e unica e memoravel |
| Nonsense | [ ] | [ ] Absurdo e intencional, nao parece erro |
| Onomatopeia | [ ] | [ ] Sons tornam a cena CENICA |
| Parodia | [ ] | [ ] Revelei fonte (parodia ≠ plagio) |
| Personificacao | [ ] | [ ] Objeto tem voz/emocao crivel |
| Ponto de Vista | [ ] | [ ] POV e EXTRAORDINARIO (nao o obvio) |
| Raciocinio Improvavel | [ ] | [ ] Conta mirabolante mas a logica fecha |
| Sarcasmo/Ironia | [ ] | [ ] Tom revela a verdade — nao e ambiguo |
| Setup/Punchline | [ ] | [ ] Escrevi serio primeiro, achei humor depois |
| Triade Comica | [ ] | [ ] 2 previsiveis + 1 surpresa |

- [ ] Texto reescrito tem personalidade e marca
- [ ] NAO esta "muito vendedor" — elementos suavizam

**GATE:** Sem elementos literarios aplicados = copy de promessa, nao Light Copy. A diferenca e a arte.

> Fonte: [SOURCE: M03 > Linguagem fantasiosa]

### Step 3.2: Verificar Equilibrio

- [ ] Texto mantém objetivo comercial (nao virou so entretenimento)
- [ ] Texto tem CTA ou premissa final que conecta a venda
- [ ] Se muito vendedor → adicionei mais elementos literarios
- [ ] Se muito artistico → adicionei conexao com venda
- [ ] Equilibrio: performance COM arte

> Fonte: [SOURCE: M03 > A base do Light Copy]

## Parte 4: Finalizacao

### Step 4.1: Revisar Anti-Patterns

- [ ] Verifiquei anti-pattern de CADA elemento usado (tabela acima)
- [ ] Hiperbole: exagero e intencional e obvio?
- [ ] Autodepreciacao: tem autoridade antes?
- [ ] Easter Egg: NAO expliquei a referencia?
- [ ] Desfecho Inesperado: NAO anunciei a surpresa?
- [ ] Confianca Extrema: removi "eu acho", "talvez", "quem sabe"?

> Fonte: [SOURCE: M03 > Hiperbole] + [SOURCE: M03 > Autodepreciacao]

### Step 4.2: Ler em Voz Alta e Testar [GATE]

- [ ] Li texto em VOZ ALTA completo
- [ ] Ritmo e fluidez OK (nao trava ao ler)
- [ ] Li para OUTRA PESSOA
- [ ] Reacao do ouvinte (marque ao menos uma):
  - [ ] Riu
  - [ ] Se emocionou
  - [ ] Disse "como voce pensou nisso?"
  - [ ] Outra reacao positiva: _______________
- [ ] Se nao houve reacao → elementos estao fracos → voltei ao Step 3.1

**GATE:** Sem ler em voz alta = nao publicar. Se ouvinte nao reage, faltam elementos ou estao fracos.

> Fonte: [SOURCE: M03 > Setup e Punchline] + [SOURCE: M02 > Revisao de Copy]

## Regras de Decisao — Referencia Rapida

| ID | Condicao | Acao |
|----|----------|------|
| RD-01 | Premissa revela intencao de venda | Reescreva sem revelar o produto |
| RD-02 | Conversao → gancho visual / Viralidade → antitese / Venda → confianca | Selecione elementos por contexto |
| RD-03 | Copy agressiva demais (muito vendedor) | Adicione elementos literarios para suavizar |
| RD-04 | Perdeu conexao com venda (muito artistico) | Adicione CTA ou premissa final |
| RD-05 | Exagero nao parece intencional | Torne a hiperbole ESCARRADA |
| RD-06 | So autoridade sem empatia | Balanceie com autodepreciacao |
| RD-07 | Perspectiva e a obvvia | Busque POV extraordinario |
| RD-08 | Tentando ser engracado sem base | Escreva serio primeiro, ache o humor depois |
