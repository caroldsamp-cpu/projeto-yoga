## Parte 1: Preparacao

### Step 1.1: Briefing de Copy [GATE]

- [ ] Nome descritivo da peca definido
- [ ] Codigo interno atribuido (ex: L10-VTSD-EMAIL-AQ01)
- [ ] Data de entrega definida
- [ ] Objetivo de MARKETING definido (o que quero que a pessoa FACA)
- [ ] Objetivo de COMUNICACAO definido (qual problema de comunicacao estou resolvendo)
- [ ] Publico definido (demografia + comportamento + temperatura)
- [ ] Formato definido (email, VSL, reels, ads, etc.)
- [ ] Pesquisa & referencias coletadas
- [ ] Big Idea (Bigadia) definida — UMA ideia central unica

**GATE:** Sem briefing = sem copy. Excecao apenas para criacoes rotineiras ja dominadas.

> Fonte: [SOURCE: M02 > Briefing de Copy]

### Step 1.2: Avatar Mapeado

- [ ] Ficha demografica preenchida (nome, idade, genero, renda, foto, etc.)
- [ ] Mapa de empatia preenchido (VE, OUVE, SENTE, FAZ, DORES, GANHOS)
- [ ] Linguagem do avatar identificada (jargao, formalidade, emojis)
- [ ] OPCIONAL: Multiplos avatares separados em tabelas distintas

> Fonte: [SOURCE: M02 > Estudo do Avatar]

### Step 1.3: Emocao Dominante [GATE]

- [ ] Escolhi: esta peca usa **MEDO** ou **GANANCIA** (marque um)
- [ ] Confirmei: NAO vou misturar as duas emocoes na mesma peca
- [ ] Apliquei Maslow para encontrar medos/desejos adicionais
- [ ] OPCIONAL: Pesquisei em Answer the Public, YouTube autocomplete, comentarios

**GATE:** Misturar medo e ganancia = efeito neutro. Escolha UMA.

> Fonte: [SOURCE: M02 > Emocoes da Persuasao]

### Step 1.4: Temperatura do Publico

- [ ] Classifiquei: publico e **FRIO**, **MORNO** ou **QUENTE**
- [ ] Selecionei tipo de lead adequado a temperatura:
  - [ ] Frio → Indireto / Noticia / Pergunta
  - [ ] Morno → Pergunta / Indireto / Noticia
  - [ ] Quente → Direto / Empatia / Promessa
- [ ] Confirmei: NAO estou usando lead direto para publico frio

> Fonte: [SOURCE: M02 > Termometro de Aquecimento]

## Parte 2: Construcao

### Step 2.1: Headline

- [ ] Escrevi 3-5 opcoes de headline
- [ ] Tipo de headline adequado a temperatura (curiosidade para frio, direto para quente)
- [ ] Headline e ESPECIFICA (nao generica)
- [ ] Se historia: comeca no meio (in medias res)
- [ ] Testei: a primeira frase faz querer ler a segunda?

> Fonte: [SOURCE: M02 > Headlines e Subheadlines]

### Step 2.2: Estrutura de Texto [GATE]

- [ ] **Introducao** presente com pelo menos 1 dos 3 C's (Curiosidade, Conflito, Contexto)
- [ ] **Desenvolvimento** com argumentos, evidencias, analogias
- [ ] **Conclusao** com sintese, reflexao ou virada inesperada
- [ ] Uma unica ideia por texto (universo cognitivo unico)
- [ ] Coesao verificada (conexoes entre frases/paragrafos)
- [ ] Coerencia verificada (sequencia logica irrefutavel)

**GATE:** Estrutura > tudo. Sem estrutura, nenhum gatilho salva.

> Fonte: [SOURCE: M02 > Estrutura de Texto]

### Step 2.3: Gatilhos Mentais

- [ ] Selecionei 2-4 gatilhos complementares para esta peca
- [ ] Gatilhos aplicados com SUTILEZA (nao de forma descarada)
- [ ] Verifiquei anti-pattern de cada gatilho usado:

## Parte 3: Refinamento

### Step 3.1: Objecoes

- [ ] Identifiquei as objecoes que meu avatar JA tem (nao inventei novas)
- [ ] Para cada objecao, tenho 1+ narrativa de matanca:
  - [ ] Logica/premissas
  - [ ] Analogia
  - [ ] Historia
  - [ ] Prova
  - [ ] Psicologia reversa
  - [ ] Ancoragem
- [ ] Confirmei: NAO estou levantando objecoes que a pessoa nao tinha
- [ ] Objecoes estao no canal certo (remarketing, DM, email — nao tudo no VSL)

> Fonte: [SOURCE: M02 > Objecoes]

### Step 3.2: Oferta (se aplicavel)

- [ ] Cada entregavel tem: curiosidade + beneficio + vantagem + formato
- [ ] Bonus complementam (nao sao coisas que ja deveriam estar no produto)
- [ ] UVP (Proposta Unica de Valor) definida em 1 frase
- [ ] Garantia definida (min 7 dias legal)
- [ ] Preco com ancoragem (mostrei referencia maior antes)
- [ ] Escassez/urgencia real
- [ ] Max 5 entregaveis (3 incriveis > 18 mediocres)

> Fonte: [SOURCE: M02 > Oferta]

## Parte 4: Finalizacao

### Step 4.1: Revisao de Copy [GATE]

- [ ] Objetivo de MARKETING atingido?
- [ ] Objetivo de COMUNICACAO atingido?
- [ ] Audiencia correta? (linguagem adequada ao estagio do funil)
- [ ] Objetivo de trafego correto? (relacionamento / descoberta / conversao / remarketing)
- [ ] 3 C's presentes? (Clareza, Coesao, Consistencia)
- [ ] Texto demais? Cortei sem piedade?
- [ ] Alinhado com avatar?
- [ ] Li em VOZ ALTA
- [ ] Li para OUTRA PESSOA
- [ ] OPCIONAL: Pedi para alguem revisar minha copy

**GATE:** Nao publicar sem ler em voz alta. Repeticao de palavra 3x = nao leu em voz alta.

> Fonte: [SOURCE: M02 > Revisao de Copy]

### Step 4.2: Publicar

- [ ] Copy publicada no mundo real (nao apenas rascunho)
- [ ] Mantendo volume consistente (meta: ___/semana)

> Fonte: [SOURCE: M02 > Lei dos Grandes Numeros]

## Parte 5: Suporte (quando aplicavel)

### Step 5.1: Reuniao de Criacao (OPCIONAL)

- [ ] Participantes: 4-8 pessoas + 1 relator
- [ ] Duracao: 10-30 min MAX
- [ ] Regra comunicada: PROIBIDO julgar ideias durante
- [ ] Pelo menos 3 tecnicas diferentes usadas
- [ ] Lista de ideias documentada pelo relator

> Fonte: [SOURCE: M02 > Reuniao de Criacao]

### Step 5.2: Inimigo Comum (OPCIONAL)

- [ ] Tipo de inimigo definido (aberto/disfarçado/sistemico)
- [ ] Risco de polarizacao avaliado e aceito
- [ ] NAO nomeei concorrente direto sem proposito estrategico

> Fonte: [SOURCE: M02 > Inimigo Comum]

## Regras de Decisao — Referencia Rapida

| ID | Condicao | Acao |
|----|----------|------|
| RD-01 | Criacao nova/complexa | Preencha briefing completo |
| RD-02 | Fuga de dor = MEDO / Busca de prazer = GANANCIA | NUNCA misture |
| RD-03 | Frio → curiosidade / Quente → direto | Headline adequada a temperatura |
| RD-04 | Vendendo → novidade+porque+urgencia / Engajando → comunidade+curiosidade+historia | Empilhar 2-4 gatilhos |
| RD-05 | Objecao recorrente → crie narrativa / 1 vez → ignore | Matar ou ignorar |
| RD-06 | Mais de 5 entregaveis → corte | 3 incriveis > 18 mediocres |
| RD-07 | Verificavel → escassez / Digital ilimitado → urgencia | Escolher tipo correto |
| RD-08 | 1 peca nao performou → crie mais / Nenhuma → revise avatar | Volume ou pivotar |
| RD-09 | Campanha/lancamento → reuniao / Peca simples → pule | Convocar ou nao |
| RD-10 | Polarizacao controlada → inimigo com humor / Preservar relacoes → evite | Posicionar ou nao |
