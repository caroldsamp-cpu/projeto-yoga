# Quality Gate - Light Copy Rewrite

**Version:** 1.0.0
**Squad:** light-copy-kb
**Checklist de validacao para reescritas Light Copy.**

## Como usar

1. Aplique este checklist a TODA copy produzida ou revisada pelo squad
2. Items em "Veto Conditions" sao bloqueantes — copy nao passa sem resolver
3. Score minimo para aprovacao: 8/10

---

## Antes da Reescrita

- [ ] Texto original está claro e completo?
- [ ] Tipo de conteúdo identificado (email, reel, description, etc.)?
- [ ] Contexto (público-alvo, objetivo) está definido?

## Durante a Reescrita

- [ ] Pelo menos 2 técnicas Light Copy foram aplicadas?
- [ ] Cada mudança foi explicada com referência à aula?
- [ ] Manteve-se a voz da Gaby (tom acolhedor)?
- [ ] Nenhuma promessa exagerada foi adicionada?

## Após a Reescrita

- [ ] Texto flui naturalmente?
- [ ] Anotações são claras e rastreáveis?
- [ ] Técnicas aplicadas fazem sentido para o tipo de conteúdo?
- [ ] Qualidade geral >= 8/10?

## Veto Conditions

❌ BLOQUEIA se:
- Perder informação essencial do original
- Tom virar duro/vendedor (não acolhedor)
- Técnica aplicada não corresponder à aula referenciada
- Anotações faltarem ou serem genéricas

---

**Status:** Operational — criterios validados contra M1-M4 + S10X
