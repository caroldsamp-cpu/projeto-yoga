## Estrutura principal

- `.aios-core/` - core e comandos do framework AIOS
- `docs/` - PRD, stories e guias do projeto
- `.agent/`, `.claude/`, `.antigravity/` - integrações e workflows de agentes

## Comandos (a partir da raiz)

- `npm run lint`
- `npm run typecheck`
- `npm test`
- `npm run verify`

Observação: os comandos acima executam dentro de `.aios-core`.
