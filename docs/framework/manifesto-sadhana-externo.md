# Manifesto Sadhana (Externo)

Versão: 0.1  
Data: 2026-02-20

Sadhana é um caminho simples para dias reais.

Não é sobre fazer perfeito.  
É sobre voltar para você, um pouco por dia.

Quando o mundo fica barulhento, a prática te devolve presença.  
Quando tudo parece demais, a respiração abre espaço.  
Quando a rotina pesa, o possível já é suficiente.

Aqui, você não precisa provar nada.  
Você só precisa começar do jeito que dá.

No seu tempo.  
Sem comparação.  
Sem culpa.

A gente acredita em constância gentil:
1. menos intensidade vazia,
2. mais presença concreta,
3. menos promessa,
4. mais prática.

Sadhana é isso:
vida real, corpo real, processo real.
