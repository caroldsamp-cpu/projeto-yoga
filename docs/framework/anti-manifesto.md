# Anti-Manifesto Sadhana

Versão: 0.1  
Data: 2026-02-20  
Uso: guardrail editorial e de marca para `marketing-yog`.

## O que a marca NÃO é

1. Não é promessa de resultado rápido.
2. Não é linguagem de medo ou culpa.
3. Não é estética de perfeição corporal.
4. Não é comparação entre alunas.
5. Não é pressão de “você precisa”.
6. Não é marketing agressivo disfarçado de cuidado.

## Frases proibidas

1. “Se não doer, não funciona.”
2. “Transformação em 7 dias.”
3. “Corpo perfeito.”
4. “Você precisa vencer.”
5. “Última chance, só hoje.”

## Substituições obrigatórias

1. De pressão para convite:
“você precisa” -> “se fizer sentido para você”.
2. De milagre para processo:
“mudar sua vida rápido” -> “mudança gradual e consistente”.
3. De rigidez para possibilidade:
“faça assim” -> “se der, faz assim; se não der, fica nessa versão”.
4. De comparação para presença:
“melhor que antes” -> “mais presente que ontem”.

## Teste de publicação (go/no-go)

Bloquear publicação se:
1. `voice_fit` baixo,
2. houver promessa inflada,
3. houver urgência artificial,
4. houver violação de segurança emocional.

Publicar somente se:
1. utilidade prática estiver clara,
2. linguagem estiver simples e humana,
3. tom estiver fiel à voz da Gaby,
4. tese Sadhana estiver explícita.
