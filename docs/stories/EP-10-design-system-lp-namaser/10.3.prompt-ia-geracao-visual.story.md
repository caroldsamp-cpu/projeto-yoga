## Story

**As a** product owner (Gaby),
**I want** a comprehensive AI prompt that generates the full visual design of the NamaSer landing page in tools like v0 or Lovable,
**so that** I can see the actual page, iterate on it, and approve the visual before we extract the design system.

## Acceptance Criteria

1. Prompt includes complete visual specification: colors (hex values), typography (font families, sizes, weights), spacing system, border radius, shadows
2. Prompt includes all LP sections from wireframe (Story 10.2) with copy direction for each
3. Prompt specifies light/luminous aesthetic with yoga-appropriate elegance
4. Prompt includes responsive behavior instructions (mobile-first)
5. Prompt includes micro-interaction and animation directions (inspired by reference sites, adapted for calm/welcoming feel)
6. Prompt includes specific image/visual placeholders with art direction
7. Prompt includes CTA styling (colors, hover states, language)
8. Generated page is visually approved by Gaby before proceeding to Story 10.4
9. Document saved at `docs/stories/EP-10-design-system-lp-namaser/ui-prompt-lp-namaser.md`
10. If Gaby requests changes, prompt is iterated and re-generated (max 3 iterations documented)
