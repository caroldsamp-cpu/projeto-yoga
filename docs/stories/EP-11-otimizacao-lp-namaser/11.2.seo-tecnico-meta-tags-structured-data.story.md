## Story

**As a** potential student finding NamaSer through Google or social media shares,
**I want** the page to have proper SEO tags, rich previews, and structured data,
**so that** the link appears professionally when shared on WhatsApp/Instagram/Facebook and ranks better on Google.

1. Title tag with relevant keywords (copy approved by squad Light Copy + Gaby)
2. Meta description with 150-160 characters (copy approved by squad Light Copy + Gaby)
3. All OG tags implemented: og:type, og:url, og:title, og:description, og:image, og:locale
4. OG image created (1200x630) — options presented to user before deciding
5. Canonical URL pointing to `https://namaser.vercel.app/`
6. JSON-LD structured data (type: Course) with real product data
7. Validated with OG preview tool (WhatsApp/social sharing test)
8. Twitter card meta tags included (twitter:card, twitter:title, twitter:description, twitter:image)
