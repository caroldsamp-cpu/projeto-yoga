## Story

**As a** visitor accessing namaser.vercel.app,
**I want** the page to load fast with images optimized and without layout shifts,
**so that** I have a smooth first impression and don't abandon the page before seeing the content.

## Acceptance Criteria

1. Hero image without `loading="lazy"`, with `fetchpriority="high"`
2. `<link rel="preload">` for the hero image in the `<head>`
3. All images have explicit `width` and `height` attributes
4. Images converted to WebP format with jpg/png fallback using `<picture>` element
5. `font-display: swap` verified on Google Fonts link
6. LCP measured before and after changes (Lighthouse or WebPageTest) — target < 2.5s
7. No visual regression — images must look identical after optimization
8. CLS reduced (target < 0.1) by adding dimensions to all images
