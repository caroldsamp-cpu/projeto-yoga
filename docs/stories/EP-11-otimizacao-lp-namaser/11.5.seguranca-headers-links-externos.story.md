## Story

**As a** site owner,
**I want** proper security headers configured and external links protected,
**so that** the site is resistant to common web vulnerabilities and user data is protected.

## Acceptance Criteria

1. Audit of security headers comparing current state (post commit 343c36e) with analysis recommendations
2. X-Content-Type-Options: nosniff confirmed
3. X-Frame-Options: DENY confirmed
4. Referrer-Policy: strict-origin-when-cross-origin confirmed
5. Permissions-Policy configured (camera=(), microphone=())
6. All external links have `rel="noopener noreferrer"` (Hotmart, YouTube, WhatsApp, Instagram)
7. `<html>` tag has `lang="pt-BR"`
8. CSP header reviewed and updated if necessary — present options before changing CSP
