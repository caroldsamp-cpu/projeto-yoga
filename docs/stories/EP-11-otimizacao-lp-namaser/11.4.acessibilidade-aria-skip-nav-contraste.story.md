## Story

**As a** visitor using assistive technology (screen reader, keyboard navigation),
**I want** the page to have proper ARIA roles, skip navigation, and adequate contrast,
**so that** I can navigate and understand the content regardless of my abilities.

## Acceptance Criteria

1. Carousel with `role="region"`, `aria-roledescription="carousel"`, `aria-label` on each slide
2. Carousel navigation buttons with descriptive `aria-label`
3. Carousel dots with `role="tablist"` and `aria-selected`
4. Accordion with `aria-expanded` and `aria-controls` on all items
5. Skip navigation link implemented ("Pular para o conteudo")
6. text-secondary color adjusted to contrast ratio >= 5.0:1 (WCAG AA) — color options presented to user before implementing
7. Carousel dot touch targets >= 44x44px
8. Screen reader test documented (NVDA or VoiceOver)
9. No visual regression from contrast adjustments — palette perception must remain warm and acolhedor
