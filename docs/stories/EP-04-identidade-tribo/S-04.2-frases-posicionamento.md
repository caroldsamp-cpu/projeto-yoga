# S-04.2 - Criar Frases de Posicionamento por Canal

**Epic:** EP-04 - Identidade de Tribo & Branding
**Story ID:** S-04.2
**Tipo:** Copywriting / Branding
**Prioridade:** SHOULD
**Estimativa:** 3 pontos
**Status:** [ ] Nao Iniciada
**Dependencias:** EP-01 (Framework Sadhana), S-03.1 (Light Copy Pattern)

---

## User Story

**Como** responsavel pela comunicacao do NamaSer,
**Quero** ter frases de posicionamento prontas para cada canal de comunicacao,
**Para que** a marca soe consistente em todos os pontos de contato, da bio do Instagram ao email de boas-vindas.

---

## Objetivo

Criar um banco de frases de posicionamento adaptadas para cada canal onde o NamaSer se comunica: Instagram (bio, destaques), WhatsApp (mensagens de grupo, mensagens individuais), email (assinatura, headers), landing page (headline, subheadline, CTAs) e outros pontos de contato. Cada frase deve comunicar a essencia do Sadhana no formato e tom adequados ao canal.

---

## Contexto

Uma marca forte nao muda o que diz - muda COMO diz dependendo do canal. A mesma essencia ("presenca real no caos urbano") precisa soar diferente em 150 caracteres de bio do Instagram vs um paragrafo de email de boas-vindas. Este documento garante consistencia de mensagem com flexibilidade de formato.

**Sentenca central do Sadhana:**
> "Sadhana e um protocolo de presenca que utiliza ritual continuo de baixa friccao para transformar o cotidiano caotico em caminho de autotransformacao."

**Posicionamento externo:**
> "Para mulheres que nao fogem da vida - que querem trazer presenca real, aqui, agora, em 5 minutos que mudam tudo."

**Posicionamento interno (comunidade):**
> "Nos somos alquimistas do cotidiano. Nos transformamos estresse em consciencia, 5 minutos de cada vez."

## Tasks Breakdown

### Fase 1: Frases Core (Base)
- [ ] 1.1 Criar ELEVATOR PITCH em 3 versoes
  - [ ] 10 segundos (1 frase): essencia pura
  - [ ] 30 segundos (2-3 frases): essencia + diferencial
  - [ ] 60 segundos (1 paragrafo): essencia + diferencial + convite
  - [ ] Cada versao no tom Gaby (conversacional, sereno, sem marketes)

- [ ] 1.2 Criar 5+ TAGLINES alternativas
  - [ ] Variante poetica: "Onde a vida real encontra a presenca real"
  - [ ] Variante pratica: "Yoga em 5 minutos que cabem na sua vida"
  - [ ] Variante tribal: "Somos alquimistas do cotidiano"
  - [ ] Variante emocional: "Sentir leveza e um ritual, nao um destino"
  - [ ] Variante provocativa (light): "E se presenca nao fosse algo que voce busca, mas algo que voce ja tem?"

- [ ] 1.3 Criar FRASE-MANTRA principal do Sadhana
  - [ ] 3 opcoes para escolha final
  - [ ] Criterios: memoravel, poetica, verdadeira, unica

### Fase 2: Frases por Canal - Instagram
- [ ] 2.1 **Bio do Instagram** (150 caracteres max)
  - [ ] 3 versoes de bio completa
  - [ ] Incluir: quem e, o que faz, CTA
  - [ ] Formato: nome | posicionamento | CTA

- [ ] 2.2 **Destaques do Instagram** (titulos)
  - [ ] Nomes para cada destaque (Sadhana, Sobre, Praticas, Comunidade, etc.)
  - [ ] Descricao curta de cada destaque

- [ ] 2.3 **CTAs para posts**
  - [ ] 5+ variantes de CTA light para final de caption
  - [ ] 5+ variantes de CTA para stories (swipe up / link)
  - [ ] Todas no tom Gaby (convite, nao ordem)

### Fase 3: Frases por Canal - WhatsApp
- [ ] 3.1 **Status do WhatsApp** (1 linha)
  - [ ] 3 variantes
- [ ] 3.2 **Mensagem de boas-vindas para grupo**
  - [ ] Versao curta (3 linhas) e versao completa (1 paragrafo)
- [ ] 3.3 **Mensagem de convite para a comunidade**
  - [ ] Light copy, sem marketes, tom de conversa
  - [ ] 2 variantes (curta e media)
- [ ] 3.4 **Mensagem pos-aulao/evento**
  - [ ] Agradecimento + convite suave
  - [ ] Tom: quente, pessoal, sem pressao

### Fase 4: Frases por Canal - Email
- [ ] 4.1 **Linhas de assunto** (5+ variantes)
  - [ ] Para email de boas-vindas
  - [ ] Para email de conteudo
  - [ ] Para email de convite/oferta
  - [ ] Todas light copy (sem clickbait)

- [ ] 4.2 **Headers de email** (frase de abertura)
  - [ ] 3 variantes de abertura de email
  - [ ] Tom: pessoal, como se estivesse escrevendo para uma amiga

- [ ] 4.3 **Assinatura de email**
  - [ ] Nome + posicionamento + frase
  - [ ] 2 variantes

- [ ] 4.4 **CTAs de email**
  - [ ] 5+ variantes de texto para botao
  - [ ] Tom light: "Quero experimentar" vs "COMPRE AGORA"

### Fase 5: Frases por Canal - Landing Page
- [ ] 5.1 **Headline principal** (3 variantes)
  - [ ] Cada uma focando em um angulo: presenca, leveza, ritmo
- [ ] 5.2 **Subheadline** (3 variantes por headline)
- [ ] 5.3 **CTAs de pagina** (5+ variantes para botoes)
- [ ] 5.4 **Microcopy** (textos pequenos: labels, tooltips, confirmacoes)
  - [ ] "Seu espaco esta reservado" vs "Compra confirmada"
  - [ ] "Comece quando sentir vontade" vs "Acesse agora"
- [ ] 5.5 **Prompt de depoimentos** (perguntas para coletar testimonials)
  - [ ] 3 perguntas alinhadas com o tom Sadhana

### Fase 6: Variantes por Avatar
- [ ] 6.1 Adaptar frases-chave para ressoar com Marina
  - [ ] Foco: parar, respirar, leveza, tempo so seu
- [ ] 6.2 Adaptar frases-chave para ressoar com Julia
  - [ ] Foco: comecar, capacidade, autorizacao, simples
- [ ] 6.3 Adaptar frases-chave para ressoar com Camila
  - [ ] Foco: voltar, constancia, ciclo, ritmo, persistir

### Fase 7: Compilacao e Revisao
- [ ] 7.1 Compilar todas as frases em documento organizado por canal
- [ ] 7.2 Verificar coerencia de tom entre canais
- [ ] 7.3 Verificar que nenhuma frase tem marketes/heavy copy
- [ ] 7.4 Revisar contra vocabulario Gaby (S-03.2)
- [ ] 7.5 Formatar com indice navegavel

## Riscos e Mitigacoes

| Risco | Impacto | Probabilidade | Mitigacao |
|-------|---------|---------------|-----------|
| Frases soarem genericas | Alto | Media | Usar vocabulario especifico da Gaby |
| Tom mudar demais entre canais | Medio | Media | Sempre voltar ao elevator pitch como ancora |
| Frases nao caberem nos limites do canal | Baixo | Media | Testar limites de caracteres antes |
| Bio do Instagram nao transmitir posicionamento | Alto | Media | Testar com 3 pessoas externas |

---

## Regras Sadhana

### SEMPRE
- Manter a essencia identica entre canais (muda o formato, nao a mensagem)
- Cada frase deve soar como algo que a Gaby diria naturalmente
- Priorizar clareza e calor sobre criatividade e sacada
- CTAs devem ser convites, nunca ordens

### NUNCA
- Usar urgencia, escassez ou pressao em nenhum canal
- Criar frases que a Gaby nao reconheceria como suas
- Copiar formulas de bio/tagline de outros creators
- Prometer resultados que o produto nao entrega
