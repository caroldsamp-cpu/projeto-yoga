# S-04.5 - Criar Briefing de Templates Visuais para Instagram

**Epic:** EP-04 - Identidade de Tribo & Branding
**Story ID:** S-04.5
**Tipo:** Briefing / Design
**Prioridade:** SHOULD
**Estimativa:** 5 pontos
**Status:** [ ] Nao Iniciada
**Dependencias:** S-04.1 (Paleta Visual), S-04.2 (Frases), S-04.3 (Identidade de Tribo)

---

## User Story

**Como** responsavel pelo conteudo visual do Mente Leve no Instagram,
**Quero** ter briefings detalhados de templates visuais para todos os formatos (stories, feed, reels, carrossel),
**Para que** qualquer designer possa criar pecas alinhadas com a identidade Sadhana sem precisar de reunioes extensas.

---

## Objetivo

Criar briefings completos e detalhados de templates visuais para Instagram que um designer (ou ferramenta como Canva) possa executar com autonomia. Cada briefing especifica: formato, dimensoes, paleta, tipografia, elementos visuais, tom, exemplos de uso e regras do que NAO fazer. O objetivo nao e criar os templates - e criar os BRIEFINGS para que os templates sejam criados com fidelidade a marca.

---

## Contexto

O Instagram e o canal principal de comunicacao do Mente Leve. Cada formato (feed, stories, reels, carrossel) tem caracteristicas proprias e precisa de templates visuais dedicados. A identidade visual deve seguir a direcao definida em S-04.1 (urbano, real, nao floral/yoga) e as frases de S-04.2.

**Formatos Instagram:**
- **Feed Post:** 1080x1080 (quadrado) ou 1080x1350 (retrato 4:5)
- **Stories:** 1080x1920 (9:16)
- **Reels Cover:** 1080x1920 (9:16)
- **Carrossel:** 1080x1350 (retrato 4:5) por slide

**Categorias de conteudo que precisam de template:**
1. Reflexao/Pensamento do dia
2. Convite a pratica
3. Dica/Educacional
4. Depoimento/Social proof
5. Frase-mantra
6. Carrossel educativo
7. Capa de Reels
8. Anuncio de aulao/evento
9. Bastidores/Real life

## Tasks Breakdown

### Fase 1: Especificacoes Tecnicas Base
- [ ] 1.1 Documentar especificacoes tecnicas por formato
  - [ ] Feed Post: 1080x1080 ou 1080x1350, safe zone, margem interna
  - [ ] Stories: 1080x1920, areas de toque (top/bottom), safe zone
  - [ ] Reels Cover: 1080x1920, area visivel no grid (centro)
  - [ ] Carrossel: 1080x1350, consistencia entre slides, setas indicativas

- [ ] 1.2 Definir regras de composicao gerais
  - [ ] Hierarquia visual: titulo → subtitulo → corpo → CTA
  - [ ] Espaco em branco minimo (respiro visual)
  - [ ] Posicionamento de logo/marca (se aplicavel)
  - [ ] Regra de contraste texto-fundo

- [ ] 1.3 Definir sistema de grid para cada formato
  - [ ] Divisao em tercos (regra dos tercos)
  - [ ] Areas de foco por formato
  - [ ] Alinhamento de texto (esquerda, centro, por formato)

### Fase 2: Briefings de Templates para STORIES (3+ variantes)
- [ ] 2.1 **Template Story - Reflexao/Mantra**
  - [ ] Proposito: compartilhar frase do dia, mantra, pensamento
  - [ ] Layout: fundo solido/textura + frase central + @handle
  - [ ] Paleta: cores primarias (S-04.1)
  - [ ] Tipografia: fonte titulo grande, fonte atribuicao pequena
  - [ ] Elementos: textura sutil, sem imagens
  - [ ] Exemplo de uso: "Nem tudo precisa ser resolvido agora..."
  - [ ] Anti-exemplo: fundo com foto de praia + texto branco com sombra

- [ ] 2.2 **Template Story - Pergunta/Enquete**
  - [ ] Proposito: gerar interacao, entender a audiencia
  - [ ] Layout: pergunta no topo + enquete/slider + respiro visual
  - [ ] Paleta: cores complementares, destaque na pergunta
  - [ ] Elementos: sticker de enquete/slider integrado ao design
  - [ ] Exemplo de uso: "Quando foi a ultima vez que voce parou so pra respirar?"
  - [ ] Anti-exemplo: fundo lotado, 3 fontes diferentes, cores neon

- [ ] 2.3 **Template Story - Convite a Pratica**
  - [ ] Proposito: convidar para pratica do dia/semana
  - [ ] Layout: icone sutil + texto convite + link/swipe
  - [ ] Paleta: cor de destaque + fundo neutro
  - [ ] Elementos: seta ou indicacao de acao suave
  - [ ] Exemplo de uso: "Hoje: 7 minutos de respiracao. Vem comigo?"
  - [ ] Anti-exemplo: "NOVA AULA!! CLIQUE AGORA!!" com fonte impacto

- [ ] 2.4 **Template Story - Bastidores** (opcional)
  - [ ] Proposito: mostrar o real, Gaby no dia a dia
  - [ ] Layout: foto + overlay leve + texto curto
  - [ ] Tom: informal, imperfeito proposital, humano

### Fase 3: Briefings de Templates para FEED POST (4+ variantes)
- [ ] 3.1 **Template Feed - Frase/Mantra**
  - [ ] Formato: 1080x1080 (quadrado)
  - [ ] Layout: fundo limpo + frase central + assinatura
  - [ ] Paleta: primarias, max 2 cores
  - [ ] Tipografia: grande, legivel, impactante mas serena
  - [ ] Espaco: muito branco/respiro, minimalista
  - [ ] Exemplo de uso: "A vida e aqui e agora"
  - [ ] Anti-exemplo: mandala de fundo + 5 cores + fonte cursiva ilegivel

- [ ] 3.2 **Template Feed - Educacional/Dica**
  - [ ] Formato: 1080x1350 (retrato)
  - [ ] Layout: titulo no topo + corpo no meio + CTA no rodape
  - [ ] Paleta: primaria (fundo) + secundaria (destaque)
  - [ ] Tipografia: titulo bold + corpo regular
  - [ ] Elementos: icone sutil, linhas de separacao organicas
  - [ ] Exemplo: "3 momentos do dia para ancorar presenca"

- [ ] 3.3 **Template Feed - Depoimento/Social Proof**
  - [ ] Formato: 1080x1080 ou 1080x1350
  - [ ] Layout: aspas + depoimento + nome/avatar
  - [ ] Paleta: tons quentes, acolhedores
  - [ ] Tom: real, nao produzido, autentico
  - [ ] Sem foto de rosto (ou com foto real, nao stock)

- [ ] 3.4 **Template Feed - Convite/Evento**
  - [ ] Formato: 1080x1350
  - [ ] Layout: headline + detalhes + CTA
  - [ ] Paleta: cor de destaque + fundo neutro
  - [ ] Tom: informativo mas caloroso (nao marketes)
  - [ ] Exemplo: convite para aulao, abertura de turma

- [ ] 3.5 **Template Feed - Bastidores/Real Life** (opcional)
  - [ ] Formato: 1080x1350
  - [ ] Layout: foto real + texto overlay minimo
  - [ ] Tom: organico, imperfeito, humano

### Fase 4: Briefings de Templates para CARROSSEL (2+ variantes)
- [ ] 4.1 **Template Carrossel - Educativo (5-7 slides)**
  - [ ] Slide 1 (Capa): titulo chamativo + subtitulo + visual forte
  - [ ] Slides 2-5 (Conteudo): 1 conceito por slide, texto + icone
  - [ ] Slide 6 (Encerramento): resumo ou mantra + CTA
  - [ ] Slide 7 (CTA): "Salva pra lembrar" ou "Compartilha com quem precisa"
  - [ ] Consistencia: mesma paleta, mesma tipografia, setas/indicacao de swipe
  - [ ] Transicao visual: progressao de cor suave entre slides

- [ ] 4.2 **Template Carrossel - Reflexao/Narrativa (3-5 slides)**
  - [ ] Slide 1: gancho (pergunta ou observacao)
  - [ ] Slides 2-3: desenvolvimento (reflexao, premissa)
  - [ ] Slide 4: virada (reframe, insight)
  - [ ] Slide 5: encerramento (mantra + convite)
  - [ ] Tom: mais poetico, menos didatico

### Fase 5: Briefings de Templates para REELS COVER (2+ variantes)
- [ ] 5.1 **Template Reel Cover - Padrao**
  - [ ] Layout: titulo centralizado + area segura para preview no grid
  - [ ] Paleta: cor solida ou gradiente suave
  - [ ] Tipografia: grande, legivel em miniatura
  - [ ] Consistencia: todos os reels com mesmo estilo de cover (feed limpo)
  - [ ] Exemplo: "3 respiracoes para acalmar a mente"

- [ ] 5.2 **Template Reel Cover - Variante com Foto**
  - [ ] Layout: foto + overlay + titulo
  - [ ] Foto: frame do video ou foto real
  - [ ] Overlay: gradiente ou cor semi-transparente

### Fase 6: Anti-Exemplos e Guia de Adaptacao
- [ ] 6.1 Criar secao "O Que NAO Fazer" (10+ anti-exemplos)
  - [ ] NAO: mandalas, lotus, imagens de praia
  - [ ] NAO: gradientes neon, cores saturadas
  - [ ] NAO: fontes cursivas ilegíveis, fontes impacto
  - [ ] NAO: muito texto por slide/post
  - [ ] NAO: fotos stock de yoga (mulher bonita meditando na montanha)
  - [ ] NAO: urgencia visual (vermelho, exclamacoes, CAPS LOCK)
  - [ ] NAO: muitos elementos visuais ao mesmo tempo
  - [ ] NAO: inconsistencia de paleta entre posts
  - [ ] NAO: filtros pesados que eliminam naturalidade
  - [ ] NAO: texto sem respiro (tudo junto, sem margem)

- [ ] 6.2 Criar guia de adaptacao
  - [ ] Como trocar texto mantendo layout
  - [ ] Como adaptar cores dentro da paleta
  - [ ] Como personalizar por avatar (tom ligeiramente diferente)
  - [ ] O que pode mudar vs o que NUNCA muda

### Fase 7: Compilacao Final
- [ ] 7.1 Organizar todos os briefings em documento unico
- [ ] 7.2 Incluir especificacoes tecnicas como anexo
- [ ] 7.3 Criar indice navegavel
- [ ] 7.4 Incluir resumo executivo
- [ ] 7.5 Revisar coerencia com S-04.1 (Paleta Visual)
- [ ] 7.6 Validar que briefings sao suficientes para designer externo trabalhar

## Regras Sadhana

### SEMPRE
- Visual minimalista com respiro (menos e mais)
- Cores que transmitam calma E modernidade
- Tipografia legivel em todos os tamanhos
- Consistencia visual entre todos os formatos
- Real > perfeito (estetica humana, nao artificial)

### NUNCA
- Estetica yoga padrao (mandalas, lotus, pastel floral)
- Excesso de elementos visuais
- Fontes ilegíveis ou muito decorativas
- Urgencia visual (vermelho, CAPS, exclamacoes)
- Inconsistencia de paleta entre posts
