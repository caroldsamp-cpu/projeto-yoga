# S-04.1 - Definir Paleta de Cores e Elementos Visuais

**Epic:** EP-04 - Identidade de Tribo & Branding
**Story ID:** S-04.1
**Tipo:** Branding / Direcao de Arte

**Status:** [ ] Nao Iniciada

## User Story

**Como** responsavel pela identidade visual do NamaSer,
**Quero** ter uma paleta de cores, tipografia e direcao de arte definidas,
**Para que** toda comunicacao visual reforce o posicionamento urbano, real e acessivel da marca.

---

## Objetivo

Definir a identidade visual do NamaSer / Sadhana que seja coerente com o posicionamento: yoga URBANO, vida REAL, presenca no CAOS. A identidade visual deve se diferenciar radicalmente do padrao yoga (floral, pastel, mandalas, natureza) e comunicar modernidade, acessibilidade e autenticidade.

---

## Contexto

A maioria das marcas de yoga usa paletas florais, tons pasteis suaves, imagens de natureza, mandalas e uma estetica "zen". Isso cria uma barreira para o publico-alvo do NamaSer - mulheres urbanas de 25-40 anos que nao se identificam com essa estetica porque acham que "yoga nao e pra elas".

**O posicionamento Sadhana exige:**
- Visual URBANO (cidade, concreto, cotidiano)
- Cores que transmitam CALMA sem parecer "alternativo"
- Estetica MODERNA que poderia ser de uma marca de lifestyle
- Zero mandalas, zero lÃ³tus, zero imagens de praia/montanha
- Elementos que falem de VIDA REAL (cozinha, escritorio, transporte)

**Referencia:** A Gaby faz yoga em casa, no dia a dia, com roupas normais. A identidade visual deve refletir isso.

---

## Scope

### Inclui
- Paleta de cores primaria (3-4 cores)
- Paleta de cores secundaria/complementar (2-3 cores)
- Cores de texto e fundos
- Direcao de tipografia (familias de fontes recomendadas)
- Direcao de fotografia (o que mostrar, o que NAO mostrar)
- Elementos graficos (padrao, texturas, iconografia)
- Moodboard descritivo (referencias visuais)
- Anti-moodboard (o que NAO fazer, com exemplos)
- Guia de aplicacao por canal (Instagram, WhatsApp, email)

### Nao Inclui
- Criacao de logo (fora do scope deste projeto)
- Templates prontos no Canva/Figma (isso e S-04.5)
- Criacao de pecas visuais finais
- Contratacao de designer/fotografo

## Tasks Breakdown

### Fase 1: Pesquisa Visual e Referencias
- [ ] 1.1 Pesquisar 10+ marcas de yoga com visual "padrao" (o que NAO queremos)
  - [ ] Listar elementos visuais comuns (mandalas, lotus, pastel)
  - [ ] Documentar por que nao funciona para o posicionamento Sadhana

- [ ] 1.2 Pesquisar 10+ marcas de lifestyle/wellness com visual urbano
  - [ ] Lululemon, Calm, Headspace, Nike Training, Peloton
  - [ ] Extrair elementos visuais que comunicam "acessivel + moderno"

- [ ] 1.3 Pesquisar 5+ marcas brasileiras com visual minimalista e humano
  - [ ] Extrair o que funciona para o publico feminino 25-40

- [ ] 1.4 Compilar moodboard descritivo (palavras-chave visuais)
  - [ ] Atmosfera: urbano, cotidiano, real, luz natural, imperfeicao bonita
  - [ ] Cores: tons terrosos, neutrals quentes, acentos profundos
  - [ ] Texturas: concreto, madeira, tecido, pele real

### Fase 2: Paleta de Cores
- [ ] 2.1 Definir paleta PRIMARIA (3-4 cores)
  - [ ] Cor base (fundo principal): tom neutro quente
  - [ ] Cor principal (identidade): tom terroso ou profundo
  - [ ] Cor de destaque (CTA, destaques): tom vibrante mas sereno
  - [ ] Cor de texto: escuro mas nao preto puro
  - [ ] Especificar HEX, RGB e nome descritivo de cada cor

- [ ] 2.2 Definir paleta SECUNDARIA (2-3 cores)
  - [ ] Cor complementar 1: para variacoes e fundos secundarios
  - [ ] Cor complementar 2: para graficos e elementos decorativos
  - [ ] Cor de alerta/emocional: para momentos de destaque
  - [ ] Especificar HEX, RGB e nome descritivo

- [ ] 2.3 Definir regras de uso das cores
  - [ ] Proporcao recomendada (60-30-10 ou similar)
  - [ ] Combinacoes permitidas e proibidas
  - [ ] Contraste minimo para acessibilidade de leitura
  - [ ] Cores para fundos claros vs fundos escuros

- [ ] 2.4 Criar demonstracao de aplicacao
  - [ ] Como ficaria em post do Instagram
  - [ ] Como ficaria em story
  - [ ] Como ficaria em email

### Fase 3: Tipografia
- [ ] 3.1 Definir familia tipografica principal (titulos e destaques)
  - [ ] Preferencia: sans-serif humanista ou serif moderna
  - [ ] Criterios: legibilidade, personalidade, disponibilidade
  - [ ] 2-3 opcoes recomendadas (fontes gratuitas de preferencia)

- [ ] 3.2 Definir familia tipografica secundaria (corpo de texto)
  - [ ] Criterios: leitura confortavel em tela, acessibilidade
  - [ ] 2-3 opcoes recomendadas

- [ ] 3.3 Definir hierarquia tipografica
  - [ ] Tamanhos para titulo, subtitulo, corpo, caption
  - [ ] Pesos (bold, regular, light) e quando usar cada um
  - [ ] Espacamento entre linhas e paragrafos

### Fase 4: Direcao de Fotografia
- [ ] 4.1 Definir o que MOSTRAR (direcao positiva)
  - [ ] Ambientes: sala de estar, cozinha, escritorio, varanda urbana
  - [ ] Pessoas: mulheres reais, roupas do dia a dia, corpos diversos
  - [ ] Acoes: praticando em casa, no chao da sala, entre tarefas
  - [ ] Iluminacao: luz natural, suave, dourada (golden hour indoor)
  - [ ] Emocoes: serenidade, leve sorriso, olhos fechados, respiro

- [ ] 4.2 Definir o que NAO mostrar (anti-direcao)
  - [ ] NAO: estudio perfeito, roupas de yoga caras, corpos "perfeitos"
  - [ ] NAO: praia, montanha, cachoeira, natureza idealizada
  - [ ] NAO: posturas avancadas, contorcoes, acrobacias
  - [ ] NAO: filtros pesados, pele sem textura, cenarios artificiais
  - [ ] NAO: mandalas, incenso, cristais (cliches yoga)

- [ ] 4.3 Criar guia de enquadramento
  - [ ] Close: rostos, maos, detalhes do corpo em movimento
  - [ ] Medio: pessoa em contexto real (sala, cozinha)
  - [ ] Aberto: ambiente com pessoa integrada (nao posando)

### Fase 5: Elementos Graficos
- [ ] 5.1 Definir estilo de iconografia
  - [ ] Minimalista, tracos finos, organico (nao geometrico duro)
  - [ ] Icones para: respiracao, movimento, pausa, ciclo, presenca

- [ ] 5.2 Definir texturas e padroes
  - [ ] Textura de papel (imperfeicao intencional)
  - [ ] Linhas organicas (nao retas perfeitas)
  - [ ] Gradientes suaves (transicoes de cor)

- [ ] 5.3 Definir tratamento de imagens
  - [ ] Filtro padrao (tonalidade, saturacao, contraste)
  - [ ] Overlay permitidos (cor, gradiente leve)
  - [ ] Brilho e calor recomendados

### Fase 6: Anti-Moodboard e Formatacao
- [ ] 6.1 Criar anti-moodboard completo
  - [ ] 10+ exemplos do que NAO fazer (com justificativa)
  - [ ] Cliches de yoga a evitar
  - [ ] Tendencias visuais que nao combinam

- [ ] 6.2 Criar guia de aplicacao por canal
  - [ ] Instagram Feed: proporcoes, cores, tipografia
  - [ ] Instagram Stories: proporcoes, cores, tipografia
  - [ ] WhatsApp: formatacao de mensagens
  - [ ] Email: header, corpo, footer

- [ ] 6.3 Formatar documento final com indice
- [ ] 6.4 Incluir resumo executivo (1 pagina)

## Regras Sadhana

### SEMPRE
- Priorizar o REAL sobre o idealizado
- Cores que transmitam calma E modernidade simultaneamente
- Visual que uma mulher urbana de 30 anos se identifique
- Acessibilidade e legibilidade em primeiro lugar

### NUNCA
- Usar estetica "yoga tradicional" (mandalas, lÃ³tus, pasteis)
- Criar visual elitista ou inacessivel
- Usar filtros que escondam a realidade
- Copiar identidade de outra marca diretamente
