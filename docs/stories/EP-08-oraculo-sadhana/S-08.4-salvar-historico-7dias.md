## User Story
Como **aluna recorrente**, quero **abrir o site e receber uma carta aleatória sem cadastro e sem salvar nada**, para que **o ritual seja leve e imediato**.

## Objetivo
Garantir uma experiência principal simples, rápida e sem persistência no MVP.

## Tasks Breakdown
- [ ] Definir regra explícita de não persistência por usuário no MVP
- [ ] Validar fluxo stateless ponta a ponta
- [ ] Definir mensagens de UX para ausência de histórico
- [ ] Garantir tom acolhedor e sem linguagem de culpa

## Acceptance Criteria
### Must Have
- [ ] Fluxo funciona sem login e sem salvar dados de usuário
- [ ] Usuária consegue tirar carta com 1 clique
- [ ] Texto de apoio deixa claro que o foco é prática do dia

### Should Have
- [ ] Documento de transição para fase com histórico
