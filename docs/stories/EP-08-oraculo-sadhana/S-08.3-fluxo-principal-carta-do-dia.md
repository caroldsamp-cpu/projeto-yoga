## User Story
Como **aluna do NamaSer**, quero **tirar minha carta do dia com um clique e receber uma orientacao pratica imediata**, para que **eu consiga entrar em presenca em poucos minutos**.

## Objetivo
Implementar a experiencia principal do MVP.

## Tasks Breakdown
- [ ] Construir tela inicial com CTA "Tirar carta"
- [ ] Implementar sorteio de carta ativa
- [ ] Exibir blocos: virtude, mensagem, micro pratica, pergunta
- [ ] Implementar estado de loading e fallback de erro
- [ ] Garantir responsividade mobile

## Acceptance Criteria
### Must Have
- [ ] Usuario tira carta com 1 clique
- [ ] Conteudo aparece completo e legivel
- [ ] Funciona em mobile e desktop
